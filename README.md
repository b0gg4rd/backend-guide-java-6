# Backend Guide for Java 6

A reference for backend Java 6 with CXF, Spring and MyBatis.

## Requirements

- JDK 1.6 update 45
- Apache Maven 3.2.5

## Building

For **CLI** use the follow common commands:

- `$ mvn clean package -Dmaven.test.skip=true`
- `$ mvn clean test`
- `$ mvn clean verify -P dev`
