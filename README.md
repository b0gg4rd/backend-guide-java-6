# Backend Guide for Java

A reference for backend Java with CXF, Spring and MyBatis.

## Requirements

- JDK 1.8 update 65
- Apache Maven 3.3.9

## Building

For CLI use the follow common commands:

- `$ mvn clean install`
- `$ mvn clean install -P dev`
- `$ mvn clean install -Dmaven.test.skip=true`
